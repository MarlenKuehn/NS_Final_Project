Map Functionality Guide

▼

This interactive map visualizes network data with geographical coordinates. Here's what you can do:

  • Load Data: Upload your JSON file containing nodes (with coordinates and family attributes) and edges.
  • Node Visualization:
    • Nodes are displayed as colored circles on the map
    • Colors represent different families (see legend in bottom-right)
    • Hover over nodes to highlight their connections
  • Edge Display:
    • Toggle between "Show All Edges" and "Show Edges on Hover" modes
    • Edges become thicker when hovering over connected nodes
  • Node Information:
    • Click any node to see detailed information
    • View node ID, family, coordinates, and number of connections
    • Click on neighbor IDs in the popup to navigate to those nodes
  • Navigation:
    • Pan and zoom the map freely
    • Click neighbor links to automatically fly to that location
